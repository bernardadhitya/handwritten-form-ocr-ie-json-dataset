# Description
This dataset is created from selecting signature images from [yolo x image](https://www.kaggle.com/datasets/medali1992/cheque-images?select=yolo+x+image)

# Content
There are cheques from 10 banks in this dataset:
1. Attihari
2. Axis
3. Banque
4. Biat
5. Canara
6. Credit
7. HSBC
8. ICICI
9. Saudi
10. universelle

Each bank has ~100 cheque images (*already processed*)
